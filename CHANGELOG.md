# UnitMode Changelog

## v0.1.0 (unreleased)

- Initial release targeting WoW Classic Era (1.15.x)
- 13 built-in WC3 unit presets across 4 factions (Alliance, Horde, Undead, Night Elf)
- Action bar overlay lockdown — 4 core slots + 1 ultimate, all other bars hidden
- Custom unit builder with UMv1 export/import strings (sharable like WeakAuras)
- WC3-style portrait overlay with class icon and faction color accent
- Per-character SavedVariables
- Slash commands: `/unitmode` (or `/um`) — opens panel; `enable`, `disable`, `reset`, `status`
